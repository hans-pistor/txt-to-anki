Overview
--------

txt-to-anki is a command-line tool that helps you convert plain text files into Anki deck formats,
making it easy to create flashcards for spaced repetition learning from your text content.

Features
--------

* Simple command-line interface
* Convert text files to Anki deck format
* Type-safe implementation with full mypy support
* Comprehensive test coverage

Quick Start
-----------

Install using pipx (recommended):

.. code-block:: bash

   pipx install txt-to-anki

Or using pip:

.. code-block:: bash

   pip install txt-to-anki

Convert a text file:

.. code-block:: bash

   txt-to-anki
