# Project Milestones

## M1 – Text Pipeline Skeleton
- Implement CLI scaffolding and configuration parsing.
- Provide plain English sentence segmentation and simple filters.
- Support CSV export as a first integration point.

## M2 – Anki Export
- Integrate `genanki` to produce `.apkg` files.
- Add source metadata to cards.

## M3 – Language Profiles
- Introduce language profile abstraction.
- Add Japanese support with appropriate segmentation tools.

## M4 – Enhancements
- Implement translation hooks and deck preview tooling.
- Support random sampling and frequency-based filtering.

## M5 – Polishing
- Finalize documentation, packaging, and continuous integration.
- Expand automated test coverage across the pipeline.
